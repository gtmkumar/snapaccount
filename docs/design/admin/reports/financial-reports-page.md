# Financial Reports Page — Design Spec (Admin)

> Phase 6F · Track F3 · Path: `/reports` · Roles: ADMIN, CA, LOAN_OFFICER, OPS (read).
> UI design specification (Phase 6F deliverable owned by ui-ux-agent). The implementing component is `ReportsPage.tsx`.

## 1. Purpose
Generate, preview, download, and share-link the six standard accounting/tax report types. Reuses `PdfViewer` (Phase 6B) and the share-with-CA / share-with-bank workflow primitives from Phase 6C.

## 2. User goal
"As a CA, I want to generate a Trial Balance for FY 26-27 Q1, preview it, then send a signed share-link to a partner bank."

## 3. Output types
1. TrialBalance — by period.
2. P&L (Profit & Loss) — by period.
3. BS (Balance Sheet) — as-of date.
4. CashFlow — by period (operating / investing / financing).
5. TaxLiability — by FY (GST + Income Tax breakdown).
6. LedgerByAccount — by account + period.

## 4. Layout (desktop ≥ 1024px)

Three-pane structure:
- Left rail (240px): vertical Tabs with the six output types.
- Center pane (flex-1): generation form + KpiStrip of last-generated.
- Right pane (480px, collapsible): PdfViewer preview pane.

Below 1024px → tabs collapse to top horizontal scroll; preview becomes a sheet.

### 4.1 Output-type list (left rail)
Each row: icon + name + one-line description. Selected = `--brand-500` left border + bg `--surface-sunken`.

### 4.2 Generation form (center pane, top)
Adapts per type:
- Common: Org (combobox if scope > 1), Output format (PDF / JSON-preview), Title override.
- Period (TrialBalance, P&L, CashFlow, LedgerByAccount): DateRangePicker, FY-aware.
- As-of date (BS only): single-date picker.
- Account (LedgerByAccount only): Combobox.
- Comparative (P&L, BS, CashFlow): checkbox "Show prior period column".
- Currency display: chip group ₹ exact / ₹ Lakhs / ₹ Crores. Default = exact.
- Footer: Generate primary CTA. Hint "Usually < 5s".

### 4.3 KpiStrip (just below form)
After successful generation: Last generated at (DD/MM/YYYY HH:mm), File size, Pages, Status chip (Generated / Generating / Failed). Quick actions: Download PDF, Copy share-link, Email to CA, Email to Bank.

### 4.4 Recent generations (table)
Compact DataTable: Date · Type · Period · Title · Size · Generated by · Actions.
Row actions: Open in preview pane, Download PDF, Copy share-link (with expiry chip), Revoke link, Delete (ADMIN, soft).

### 4.5 PdfViewer (right pane / sheet)
Reuses Phase 6B `PdfViewer`: page nav, zoom, download, print, text selection. Top toolbar adds: "Share with CA" / "Share with Bank" buttons. Watermark badge if package is signed/watermarked: "Watermark intact".

## 5. Share-link modal

Dialog (Wide variant) with two tabs:
1. Share with CA — Combobox: pick CA (org-scoped). Expiry chips: 24h · 7d · 30d · No expiry. Optional message. Generate link → copy to clipboard + email CA.
2. Share with Bank — Combobox: partner bank (Phase 6C registry). Expiry: 24h · 7d (compliance cap). Include disclaimer (default true; canonical Phase 6C disclaimer auto-appended). Send via bank adapter (REST/Email/OAuth — same primitive as 6C).

Both surfaces show DisclaimerCard (warning tone) with the canonical Phase 6C copy, verified in dark + light.

## 6. Empty / loading / error
- No generations yet: `empty.reports` ("Generate your first report") + CTA "New Trial Balance".
- Loading generation: ProgressIndicator on form button + skeleton in preview pane.
- Generation error: Toast + inline error card "Couldn't generate. Try a smaller period." with retry.
- Preview load error: PdfViewer error state with retry.

## 7. Role gating
- ADMIN: all types, all orgs, all actions including delete + revoke.
- CA: all types within assigned orgs; share-with-bank requires admin grant; can delete own generations.
- LOAN_OFFICER: limited to TrialBalance, P&L, BS — primarily for share-with-bank.
- OPS: read-only — RoleGuard fallback notice "Generation requires CA or Admin role".

## 8. Keyboard shortcuts
- `g r` lands here.
- `n` focuses generate form.
- `Enter` (in form) submits.
- `j/k` navigates output types in left rail.

## 9. Accessibility
- Form fields all labeled; required marker on date inputs.
- Preview pane heading "Preview · {{title}}".
- Download/share buttons have descriptive `aria-label`.
- Live region announces "Report generated, {{pages}} pages".

## 10. Responsive
- < 1024px: left rail collapses into top horizontal tabs; preview becomes bottom sheet.
- < 768px: only the form + recent table visible; preview opens full-screen modal.
